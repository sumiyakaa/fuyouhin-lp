## デザイン品質基準（絶対遵守）

- 全幅（100%幅）使用。中央寄せ＋左右余白大のレイアウト禁止
- 写真・イラストがページ面積の50%以上を占めること
- セクション間で同じレイアウトパターンを連続使用しない（最低5種類のパターン使用）
- テキストだけのセクションは禁止。必ず画像・装飾・アイコンを伴う
- Heroのメインコピーは画面の60%近くを占める大文字
- 静止セクション禁止: CSS animation（フェードイン・スライドイン）を適用
- 非対称レイアウトを最低1箇所使用
- 4数字カウントアップグリッド禁止（AI/テンプレ感が出る）

## アニメーション仕様

- **方式**: CSS animation + Intersection Observer（GSAP不使用で軽量化）
- **Hero**: テキスト左からslideIn（0.6s ease-out）、バッジ下からpopUp（0.4s、stagger 0.1s）
- **各セクション**: スクロール連動fadeInUp（0.5s）。Intersection Observer threshold: 0.15
- **CTAボタン**: hover時 scale(1.05) + box-shadow拡大。最終CTAのみ pulse animation
- **料金カード**: hover時 translateY(-4px) + shadow強化
- **過度なアニメーション禁止**: 装飾のためのアニメーションは不要。ユーザーの視線誘導に必要な最小限

## 技術要件

- HTML5 セマンティックタグ使用（header, nav, main, section, footer）
- CSS カスタムプロパティでカラー管理
- JSは Intersection Observer のみ（外部ライブラリ不使用）
- フォント: Google Fonts（Noto Sans JP + Inter）display=swap
- 画像: loading="lazy" + width/height属性明示（CLS防止）
- SP breakpoint: 768px / タブレット: 1024px
- テキストはすべて選択可能（画像内テキスト禁止）

## SP（スマートフォン）対応の方針

- ナビゲーション: ハンバーガーメニュー不要（LP1枚のため）。ヘッダーは電話番号＋ロゴのみ固定
- Hero: テキスト → 写真の縦積み。CTAは横幅100%のブロックボタン
- カード類: 1列縦積み
- 料金プラン: 横スクロール（scroll-snap）
- ご利用の流れ: タイムライン形式（左に番号、右にテキスト）
- フォントサイズ: body 15px / h2 24px / h3 20px
- ボトムCTA固定バー: 「電話」＋「LINE」の2ボタンが画面下部に常時表示（position: fixed）

## 参考サイトとの差別化ポイント

参考サイト（クオーレ東京）は情報量で圧倒する「長尺LP」。ラフでは以下の方針で差別化:

1. **情報の凝縮**: 20セクション超 → 9セクションに凝縮。各セクションの密度を上げる
2. **モダンなデザイン**: 参考サイトはやや旧来的なデザイン。border-radius大きめ、余白ゆったり、カード影軽めのモダン路線
3. **色の統一感**: 参考サイトは青・緑・金・赤が混在。本LPはブルー基調＋アクセントオレンジに統一
4. **アニメーション**: 参考サイトはほぼ静的。スクロール連動の軽いアニメーションで動きを出す
5. **SP優先設計**: 不用品回収の検索はスマホが主。SP表示のUXを特に重視
